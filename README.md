# gradebook-analyzer
Project Title

GradeBook Analyzer – Python Mini Assignment

Project Description

GradeBook Analyzer is a Python program that reads a CSV file containing students' marks and generates useful insights.
It calculates average, median, highest, lowest marks, and also prints grade-wise summary.
The program also allows the user to analyze the file again without restarting the script.

Folder Structure

GRADEBOOK_ANALYZER/
│── gradebook.py
│── students.csv


How to Run

Open the GRADEBOOK_ANALYZER folder in VS Code.

Make sure gradebook.py and students.csv are in the same folder.

Run the program using:

python gradebook.py


When asked, enter:

students.csv


View the output in the terminal.
